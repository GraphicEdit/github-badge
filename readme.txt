GitHub Badge  
This plug-in allows you to create and place a GitHub css3 badge to your webpage on wordpress.

== Description ==
This plug-in allows you to create and place a GitHub css3 badge to your webpage on wordpress. With the help of this plug-in you create a css3 badge for your users or followers helping them to follow your GitHub account. This plug-in is very useful for GitHub users, for users who would like to promote his/her GitHub on Wordpress. 



Links: [Author Homepage](http://graphicedit.com/)

== Installation ==
1. Upload the "GitHub Badge" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode [GHB]xxxxxxx[/GHB]  the last part of the page/post
